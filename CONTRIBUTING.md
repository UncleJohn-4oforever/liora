# Contributing

## Dev setup

1. Install Node LTS, Rust, Ollama  
2. On Windows for Tauri: VS Build Tools + C++ workload  
3. `npm install`  
4. `npm run dev` or `npm run tauri dev`  

## Before PR

- `npm run build` passes  
- Do not commit `node_modules`, `dist`, `src-tauri/target`, backups, or personal chat data  
- Prefer small, focused PRs  

## Security

Report sensitive issues privately if possible. Never paste real API keys or user memory dumps into issues.
